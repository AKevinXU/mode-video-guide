一、视觉风格定位

你的目标客户是企业主/项目方，他们要的是**信任和专业**，不是情绪刺激。

#

1.1 核心风格词


专业 · 沉稳 · 高级 · 可信 · 商务

**对比**：

维度C端风格（错误）B端风格（正确）
**色调**金红高饱和深蓝+白+少量金
**字体**综艺体、艺术字思源黑体、苹方
**背景**动态特效、红包雨纯色/轻微渐变
**人物出镜**夸张表情、大喊大叫正装/商务休闲、沉稳
**数据展示**金额跳动、炫富图表、流程图、系统界面
**BGM**土嗨、节奏强烈轻电子、无歌词

#

1.2 配色方案


:root {
  --bg-primary: #0D1117;      /* 主背景：深色 */
  --bg-secondary: #161B22;    /* 次背景：卡片 */
  --border-color: #30363D;    /* 边框 */
  --text-primary: #E6EDF3;    /* 主文字：白 */
  --text-secondary: #8B949E;  /* 次文字：灰 */
  --gold: #D4A853;            /* 主金色 */
  --gold-light: #F0D78C;      /* 亮金色 */
  --blue: #58A6FF;            /* 销售员层级 */
  --green: #3FB950;           /* 经销商层级 */
  --purple: #A371F7;          /* 区域补贴 */
}

**使用规则**：

  • 背景：深色（#0D1117 或 #161B22）
  • 标题：金色（#D4A853）
  • 正文：白色（#E6EDF3）
  • 辅助：灰色（#8B949E）
  • 层级标识：蓝/绿/金/紫

二、画面构造规范

#

2.1 真人出镜画面

**推荐构图**：


┌─────────────────────────────────────┐
│                                     │
│    你（左半屏/60%）                  │
│    - 正装或商务休闲                  │
│    - 眼神看镜头偏右                  │
│    - 手势配合讲解                    │
│                                     │
│         手写板/iPad（右半屏/40%）    │
│         - 实时手写模式图             │
│         - 重点数据放大               │
│                                     │
└─────────────────────────────────────┘

**替代方案**：

  • 全屏真人+小窗数据（右上角弹出数据卡片）
  • 全屏录屏+小窗真人（左下角小窗出镜）

#

2.2 字幕设计规范

参数规范
**字体**思源黑体 Bold / 苹方
**大小**正文14-16px，标题20-24px
**颜色**白色为主，关键数字金色
**动画**逐字弹出或从底部滑入
**高亮**关键数字用【】框住，放大1.2倍
**停留**每句字幕至少停留2秒
**位置**底部居中，不遮挡人脸

#

2.3 数据可视化规范

**数字展示**：

  • 使用等宽字体或粗体
  • 关键数字用金色高亮
  • 配合"叮"音效

**图表展示**：

  • 金字塔图：逐层点亮，配升级条件
  • 圆环图：分段展开，配百分比
  • 表格：逐行填充，配对比色

**动画节奏**：

  • 每3-5秒一个"视觉锚点"
  • 数字跳动、转场特效、音效
  • 防止用户划走

三、剪辑规范

#

3.1 基础参数

参数建议值说明
视频比例9:16竖屏，移动端优先
分辨率1080×1920高清
帧率30fps流畅
时长60-90秒B端内容允许更长
格式MP4 (H.264)兼容性好

#

3.2 剪辑节奏公式

**快-慢-快-爆**：

段落剪辑节奏转场方式音效设计
开头3秒0.5秒/切硬切+缩放冲击重低音"咚"
痛点部分1秒/切左右滑动悬疑BGM渐入
拆解部分0.8秒/切图形遮罩转场轻快电子乐
结尾CTA0.5秒/切缩放+模糊倒计时+欢呼

#

3.3 转场规范

**推荐转场**：

  • 淡入淡出（最常用）
  • 直接切换（信息密度高时）
  • 轻微缩放（强调重点时）

**禁止转场**：

  • 花哨特效（旋转、翻页、3D）
  • 过多转场（每2秒一个转场）
  • 与内容风格不符的转场

#

3.4 音效设计

**必备音效**：

  • 开头：重低音鼓点（吸引注意力）
  • 数据出现：清脆"叮"（强调重点）
  • 转场：轻微"嗖"（过渡自然）
  • 结尾：倒计时"滴答"+成功音效（制造紧迫感）

**BGM选择**：

  • 类型：轻电子乐、无歌词
  • 节奏：与讲解节奏匹配
  • 音量：不压过人声（人声70%，BGM 30%）

---

四、封面设计规范

#

4.1 封面比例

  • 视频号封面：3:4（竖版）
  • 首帧画面：与封面一致

#

4.2 封面元素


┌─────────────────┐
│                 │
│   主标题（大字）  │  ← 金色粗体，占画面40%
│   分销模式      │
│   合规拆解      │
│                 │
│   副标题（小字）  │  ← 白色，补充信息
│   总拨比66%     │
│                 │
│   数据亮点（3个） │  ← 圆形图标+数字
│   5层  66%  3路径│
│                 │
│   底部CTA       │  ← "私信领取白皮书"
│                 │
└─────────────────┘

#

4.3 封面设计原则

  • **大字报风格**：文字为主，信息密度高
  • **颜色对比**：深色背景+金色/白色文字
  • **数字突出**：关键数据用最大字号
  • **避免杂乱**：不超过3种颜色，不超过5行文字

六、制作流程（单条视频2小时内完成）


第1步：写脚本（20分钟）
  → 用"黄金四段式"模板填空
  → 确定钩子、痛点、方案、CTA

第2步：准备素材（15分钟）
  → 手写板画好模式架构草图
  → 准备2-3个数据案例
  → 检查设备、灯光、背景

第3步：拍摄（30分钟）
  → 分段拍摄，每段单独录制
  → 拍3-5遍，选最自然的一遍
  → 每段结尾留0.5秒空白

第4步：粗剪（20分钟）
  → 导入素材，按脚本顺序排列
  → 删除口误、停顿
  → 调整节奏，控制时长

第5步：精修（30分钟）
  → 添加字幕，关键数字高亮
  → 添加音效、BGM
  → 调色，统一色调

第6步：封面+发布（15分钟）
  → 制作封面图（3:4比例）
  → 写标题、描述、话题标签
  → 设置评论区置顶
  → 发布+社交冷启动

**核心建议**：先从最简单的"真人出镜+手机剪辑"开始，跑通流程后再升级设备。内容质量 > 制作精度。
