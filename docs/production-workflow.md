## 一、内容形式选择：为什么必须真人出镜

### 1.1 三种形式对比

| 形式 | 算法友好度 | B端信任度 | 制作成本 | 推荐度 |
|------|-----------|----------|---------|--------|
| **真人出镜+手写板** | ★★★★★ | ★★★★★ | 中 | ⭐⭐⭐⭐⭐ |
| **录屏+画中画** | ★★★★☆ | ★★★★☆ | 低 | ⭐⭐⭐⭐ |
| **纯动画/PPT翻页** | ★★☆☆☆ | ★★☆☆☆ | 低 | ⭐⭐☆☆☆ |

### 1.2 真人出镜的画面构造

```
┌─────────────────────────────────────┐
│  左半屏（60%）      │  右半屏（40%）  │
│  你真人出镜         │  手写板/iPad   │
│  正装/商务休闲      │  实时手写模式图 │
│  背景：书架/白板    │  边写边讲      │
│  眼神看镜头偏右     │  重点数据放大   │
└─────────────────────────────────────┘
```

**关键细节**：
- 不要全屏放PPT，要让人看到你边写边讲
- 眼神看镜头偏右（模拟看客户的感觉）
- 手势配合讲解，增加动态感

## 二、60秒视频制作全流程

### 2.1 前期策划（30分钟）

**Step 1：选题确定**
- 选择"诊断式"选题（反常识+制造焦虑）
- 示例："我上周看了7个分销模式，5个都有这个致命漏洞"

**Step 2：脚本撰写**
- 使用"黄金四段式"结构
- 每段控制在15秒内
- 预留3-5个"视觉锚点"

**Step 3：素材准备**
- 手写板/平板+手写笔
- 提前画好模式架构草图
- 准备2-3个数据案例

### 2.2 拍摄执行（20分钟）

**设备清单**：
- 手机：iPhone 15 Pro或同等安卓旗舰（4K 60fps）
- 三脚架/手机支架：保证画面稳定
- 补光灯：环形灯，面部光线均匀
- 领夹麦：确保口播清晰

**拍摄要点**：
1. 先拍3-5遍，找到最自然的节奏
2. 分段拍摄，每段单独录制，后期拼接
3. 保持眼神稳定，不要频繁看提词器
4. 每段结尾留0.5秒空白，方便剪辑

### 2.3 后期剪辑（40分钟）

**推荐工具**：剪映专业版（免费，功能足够）

**剪辑参数**：
| 参数 | 建议值 |
|------|--------|
| 视频比例 | 9:16（竖屏） |
| 分辨率 | 1080×1920 |
| 帧率 | 30fps |
| 时长 | 60秒 |

**剪辑流程**：

```
第1步（5分钟）：导入素材，按脚本顺序排列
第2步（10分钟）：粗剪，删除口误、停顿，保留精华
第3步（10分钟）：添加字幕，关键数字用黄色高亮
第4步（10分钟）：添加音效（数字跳动、成功提示、转场）
第5步（5分钟）：调色，统一色调（深蓝+金）
```

**字幕规范**：
- 字体：思源黑体 Bold
- 关键数字：放大1.2倍，金色高亮
- 每句字幕停留≥2秒
- 语速与字幕同步

### 2.4 发布优化（10分钟）

**标题公式**：
```
【行业痛点词】+【数字】+【结果词】+【身份背书】
```

**描述模板**：
```
【核心钩子】+【价值承诺】+【行动号召】+【话题标签】

示例：
"你的分销模式可能随时被风控？我们拆解了66%波比的合规架构，
已帮12家客户规避违规风险。评论区留下你的行业+拨比，我帮你诊断。
#分销模式 #系统开发 #合规设计"
```

**封面设计**：
- 使用大字报风格，3:4比例
- 主标题：金色粗体，占画面50%
- 副标题：白色，补充关键信息
- 背景：深色渐变，突出文字

## 三、系列内容规划（7天赛马机制）

利用视频号的7天长效推荐机制，设计连续剧式内容：

| 集数 | 选题 | 目的 | 钩子设计 |
|------|------|------|---------|
| 第1集 | 《分销模式合规的3条红线》 | 科普引流 | "90%的模式都踩了第2条" |
| 第2集 | 《66%拨比怎么拆？4层架构详解》 | 专业背书 | "这样拆，平台不会风控" |
| 第3集 | 《分润系统自动结算的算法逻辑》 | 技术展示 | "每天节省3个人力" |
| 第4集 | 《客户案例：3个月从0到月流水千万》 | 结果证明 | "这个客户之前被坑了20万" |
| 第5集 | 《分销系统开发全流程：从需求到上线》 | 服务展示 | "源码交付，终身维护" |

**系列策略**：
- 每集结尾预告下一集内容
- 引导用户收藏/关注，等待更新
- 系列内容收藏率极高，算法持续推荐

## 四、发布后10分钟社交冷启动

### 4.1 社群转发

转发到**3-5个精准行业社群**：
- 分销/微商行业交流群
- 创业/副业交流群
- 私域流量运营群
- 你的客户群

**转发话术模板**：
```
"刚发了一条视频，讲分销模式合规设计的3个坑，
建议做模式的朋友看看。评论区可以留言诊断，
我帮大家初步看看模式有没有问题。"
```

### 4.2 朋友圈发布

**错误示范**：
"请大家帮我点赞支持一下，谢谢！"

**正确示范**：
```
"这条视频讲了分销模式合规的3个坑，
我们已帮12家客户规避了违规风险。
建议做模式的朋友看看，评论区可以留言诊断。
"
```

### 4.3 种子互动

找2-3个同行/客户：
1. 完整看完视频（提升完播率）
2. 写8字以上评论（提升评论质量分）
3. 转发到他们的朋友圈/社群（触发社交推荐）
