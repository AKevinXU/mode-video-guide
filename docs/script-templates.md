## 一、模板A：模式诊断型（引流最强）

**适用场景**：新号冷启动、破圈曝光、吸引有顾虑的客户
**核心逻辑**：不说"我的模式多好"，说"你的模式可能有问题"

### 脚本正文

```
【0-3秒】钩子
画面：你真人出镜，表情严肃，手持一份文件
字幕："我上周看了7个分销模式，5个都有这个致命漏洞"
音效：重低音"咚"

【3-10秒】痛点共鸣
画面：分屏对比，左边"违规模式"（红色警示），右边"合规模式"（绿色通过）
字幕："你的模式有没有这些问题？波比超过70%、层级超过3级、没有实物交付"
音效：悬疑BGM渐入

【10-40秒】专业拆解
画面：手写板画出4层架构图，逐层讲解
字幕："正确的架构应该是这样：会员5%→销售员20%-30%-50%循环→经销商分红池10%→代理商5%+区域补贴1%"
音效：轻快电子乐，每层"叮"

【40-55秒】解决方案
画面：你回到镜头前，展示3个原则
字幕："3个合规原则：1.总拨比≤70% 2.层级≤3级 3.必须有实物交付"
音效：成功音效

【55-60秒】CTA
画面：封面图定格，二维码弹出
字幕："评论区留下你的行业+拨比，我帮你诊断"
音效：固定Slogan
```

### 关键数据预期
- 完播率：>25%（悬念强）
- 评论率：>3%（诊断钩子）
- 转发率：>1%（合伙人转发）

## 二、模板B：系统录屏型（转化最强）

**适用场景**：展示技术能力、吸引需系统落地的客户

```

字幕："这套分润系统每天自动结算12万笔，误差率0.01%"
音效：重低音"咚"

【3-10秒】痛点
画面：对比图，左边"人工结算"（错发/漏发/纠纷），右边"自动结算"（准确/高效/透明）
字幕："你的分润还在人工算？一个月错发5万，客服电话被打爆"
音效：悬疑BGM

【10-45秒】系统展示
画面：后台录屏，展示核心功能
  - 会员管理：层级自动晋升
  - 订单系统：自动分润计算
  - 红包发放：每日0点自动发放
  - 数据看板：实时业绩统计
字幕：每个功能配简短说明
音效：轻快电子乐

【45-55秒】案例证明
画面：客户数据对比图（使用前vs使用后）
字幕："客户使用3个月后，结算效率提升300%，人力成本降低60%"
音效：成功音效

【55-60秒】CTA
画面：系统功能清单截图
字幕："私信'系统'，领取功能清单+报价参考"
音效：固定Slogan
```

### 关键数据预期
- 完播率：>20%（录屏有吸引力）
- 私信率：>2%（技术展示强）
- 转化率：>5%（高意向客户）

## 三、模板C：案例见证型（信任最强）

**适用场景**：建立信任、促进转化、影响犹豫期客户
**核心逻辑**：用真实案例证明"我们能做到"

### 脚本正文

```
【0-3秒】钩子
画面：客户授权的真实数据截图（脱敏）
字幕："帮这个客户3个月做到月流水千万，但一开始他被坑了20万"
音效：重低音"咚"

【3-15秒】背景故事
画面：客户访谈片段（或你口述）
字幕："这个客户之前找了一家服务商，模式设计不合规，系统经常崩溃，3个月亏了20万"
音效：悬疑BGM

【15-45秒】解决方案
画面：你讲解改造过程

  - 第二步：重新设计合规架构
  - 第三步：开发稳定系统
  - 第四步：上线运营支持
字幕：每步配关键数据和成果
音效：轻快电子乐

【45-55秒】结果展示
画面：改造前后的数据对比
字幕："改造后：月流水从0到千万，团队从0到500人，系统稳定运行18个月"
音效：成功音效

【55-60秒】CTA
画面：客户感谢截图/合影
字幕："如果你也有类似需求，私信'案例'，发你完整方案"
音效：固定Slogan
```

### 关键数据预期
- 完播率：>30%（故事性强）

- 私信率：>3%（信任度高）

## 七、脚本使用建议

### 7.1 组合策略

| 阶段 | 主要模板 | 辅助模板 | 目的 |
|------|---------|---------|------|
| 新号期（0-1000粉） | A + F | B | 破圈曝光，快速涨粉 |
| 成长期（1000-5000粉） | B + C | A + D | 建立信任，展示能力 |
| 成熟期（5000+粉） | D + E | B + C | 高端获客，提升客单价 |

### 7.2 定制化修改

每个模板都需要根据你的实际情况修改：
- **数据**：替换为你的真实数据（客户数、交付周期、价格）
- **案例**：替换为你的真实案例（客户名称需脱敏）
- **钩子**：根据你的目标客户调整痛点
- **CTA**：根据你的引流目标调整话术

### 7.3 测试优化

同一选题，用不同模板测试：
- 示例："分销系统开发多少钱"
  - 用模板F（快速问答）测试完播率
  - 用模板B（系统录屏）测试转化率
  - 对比数据，保留最优版本

> **核心原则**：模板是框架，内容是灵魂。同样的模板，不同的内容，效果天差地别。关键是提供真实、有价值的信息。
